A Service Worker is a special JavaScript file that runs in the background, separate from your main web page.
It does not have access to the DOM directly, but it can:

Intercept network requests (like fetch)

Cache resources (HTML, CSS, JS, images, API responses)

Work offline (show content when no internet)

Send push notifications

Do background sync
